## Coding style

| 大括号格式            | {<br />}                      |
| --------------------- | ----------------------------- |
| 类名                  | 大驼峰规则 `DataService`      |
| interface 命名        | I + 大驼峰 `IController`      |
| public 成员           | 大驼峰 `public bool IsValid;` |
| private internal 字段 | _ + 小驼峰 `_camelCasing`     |
| 普通成员              | 小驼峰 `camelCasing`          |



## Git commit style

| type         | commit    |
